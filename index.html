Logic programming is a programming paradigm which is largely based on formal logic. Any program written in a logic programming language is a set of sentences in logical form, expressing facts and rules about some problem domain

Understanding Logic

Logic is human reasoning that is conducted according to strict principles or rules. Programming logic is a set of principles that delineates how elements should be arranged so a computer can perform specific tasks.


Logical thinking, whether programming or formal, means applying principles in a disciplined manner to achieve an acceptable result.
